Span and Div

Span
Span basically propotionalizes an element inside of an attribute like a single sentence in a para and then allowing the implementer to use CSS properties on that small part such as:
This 1 sentence in this definition is only bold and a different font instead of the whole definition.
Div
Div tag basically applies an effect from CSS to everything inside of the opening closing brackets. Like if I had 3 paragraphs whose font I wanted to be the same kind I could just put it in a div tag and I would not have to add a font style to each paragraph starting tag.

Meta

Meta Tag
This tag is placed in the head of the page and not body. Metadata is data about data and represents info/data that cannot be expressed using the title or a link or some kind of styling, etc.

Iframe

Iframe
Iframe is used for embeding content from another source into an HTML document. Widely used applications are embedding ads on a website that can be images with links or even video and gif ads. This is also used in many hacking techniques where scammers will make the iframe take up the entire page and pose as another website. That is why a lot of big websites like google and banks refuse to be embedded by iframes so sensitive information does not go in the hands of wrong people.
